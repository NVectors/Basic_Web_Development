Image Format Page

This page covers the four common image formats used on the web (source: this page).

JPGs

JPG images are good for photos especially those with lots of gradients but don't allow for trasnparent pixels.

Scaling up the JPG image

PNGs

PNG images are good for anything that's not a photo or animated.

PNG images works well for icons, technical diagrams, logos, etc.

Scaling up the PNG image

SVGs

SVG is a vector-based graphics format, the images can be scaled up or down to any dimension without loss of quality.

This makes SVG images useful for responsive design and they have the same use case as a PNG image.

There is one potential issue with SVGs: for them to display consistently across browsers, you need to convert any text fields to outlines using your image editor.

Scaling up the SVG image

GIFs

GIFs are used for simple animations, they are limited in terms of color palette.

Also the transparent pixels are a binary option for GIFs, meaning you can't have semi-opaque pixels. This can make it difficult to get high levels of detail on a transparent background

Scaling up the GIF image
